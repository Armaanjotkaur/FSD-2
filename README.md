# Experiment 16 - Full Stack Testing

## Backend Testing
- Flask API
- Pytest used
- CRUD operations tested

## Frontend Testing
- React (Vite)
- Vitest + Testing Library
- Form validation tested

## CI/CD
- GitHub Actions used
- Runs backend + frontend tests automatically on push

## How to run locally

### Backend
cd Testing/Backend
pip install -r requirements.txt
pytest

### Frontend
cd Testing/Frontend
npm install
npm run test
